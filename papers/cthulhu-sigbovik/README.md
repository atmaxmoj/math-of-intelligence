# Cthulhu Existence Theorem

SIGBOVIK 2026 submission.

## Build

```bash
make        # build PDF
make view   # build and open
make clean  # clean auxiliary files
```

## Abstract

We prove that Lovecraft was mathematically correct.
Using Hutter's definition of intelligence as compression and the well-foundedness of ℕ,
we demonstrate that there necessarily exist intelligences that cannot be understood by any intelligence.
